# trias-risk-map

Small Vue-based prototype to show risk maps (data from GeoTiff files), without any server-side component.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
